## 1. 首页 (index.html)


## 7. URL 参数设计

### 7.1 collection.html 参数

```
collection.html?category=games          # 筛选：小游戏
collection.html?category=websites       # 筛选：治愈网页
collection.html?category=tools          # 筛选：互动工具
collection.html?category=tests          # 筛选：轻松测试
collection.html?tag=relax               # 筛选：放松标签
collection.html?tag=focus               # 筛选：专注标签
collection.html?tag=sleep               # 筛选：睡眠标签
collection.html?tag=happy               # 筛选：开心标签
collection.html?tag=relief              # 筛选：解压标签
```

### 7.2 quiz.html 参数

```
quiz.html?type=color                    # 情绪颜色测试
quiz.html?type=energy                   # 能量状态检测
quiz.html?result=color&score=xxx        # 颜色测试结果
quiz.html?result=energy&score=xxx       # 能量测试结果
```

## 8. 响应式布局

### 8.1 断点

| 断点 | 宽度 | 布局 |
|------|------|------|
| 移动端 | < 640px | 单列，卡片全宽 |
| 平板 | 640px - 1024px | 双列卡片 |
| 桌面 | > 1024px | 双列卡片，最大宽度680px居中 |

### 8.2 首页布局

- 移动端：呼吸动画全屏，下方堆叠内容
- 桌面：呼吸动画占60%高度，下方内容双列

### 8.3 合集页布局

- 移动端：单列卡片
- 平板/桌面：双列卡片网格

### 8.4 测评页布局

- 所有设备：单列，居中
- 选项：移动端垂直排列，桌面可水平排列
