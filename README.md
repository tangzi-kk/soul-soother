# Soul Soother - 心灵抚慰者

> 用抽象接住崩溃，用玩笑降低痛感

## 项目简介

Soul Soother 是一个面向极端情绪状态人群（自杀意念高发群体）的轻量级趣味治愈网站。我们不提供医学诊断，也不替代心理咨询，而是用 Z 世代最懂的"抽象自嘲"语言，让濒临崩溃的人在会心一笑中获得短暂的喘息和陪伴。

## 核心功能

- **今日抽象**：每日更新发疯文学/抽象文案
- **趣味测评**：用荒诞的题目检测精神状态（SBTI 风格）
- **发疯文学生成器**：输入情绪关键词，生成发疯文学
- **"想死"替代方案生成器**：当你想死的时候，看看还有什么可以先做完
- **互助卡抽取**：抽一张互助卡，给自己或给别人
- **求助贴/互助区**：今天有点撑不住？来这里发一句
- **激励滚动弹幕**：来自陌生人的陪伴
- **治愈合集**：收集了一些可能让你放松一小会儿的小游戏、治愈网页、互动工具
- **深夜急救包**：凌晨 1-5 点专属内容

## 技术栈

- 纯 HTML5 + CSS3 + JavaScript（零框架，极致轻量）
- CSS Variables + Flexbox/Grid
- localStorage（仅收藏功能）
- GitHub Pages 部署

## 快速开始

```bash
# 克隆仓库
git clone https://github.com/yourusername/soul-soother.git

# 进入目录
cd soul-soother

# 用 Live Server 启动（VS Code 插件）
# 或直接用浏览器打开 index.html
```
