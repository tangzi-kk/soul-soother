# Soul Soother - 产品需求文档

## 1. 项目概述

### 1.1 产品名称
**Soul Soother**（心灵抚慰者）

### 1.2 一句话描述
一个面向极端情绪状态人群（自杀意念高发群体）的轻量级趣味治愈网站，用Z世代最懂的"抽象自嘲"语言，让濒临崩溃的人在会心一笑中获得短暂的喘息和陪伴。

### 1.3 核心定位
- **不是医学诊断工具**：明确免责声明，不做心理评估或治疗建议
- **不是严肃心理健康平台**：走"真抽象"的幽默自嘲路线，像憨豆一样让人会心一笑
- **是"想死的时候可以随手打开笑一笑"的角落**：用最懂年轻人的梗，让极端情绪在笑声中暂时松绑
- **是"自杀边缘的缓冲带"**：不承诺治愈，只承诺"让你笑一秒钟"

### 1.4 目标用户（重新定位）

#### 核心用户：极端情绪状态人群
- **自杀意念高发群体**：经常有"不想活了""活着没意思"念头的人
- **边缘型人格特质人群**：情绪极端波动，自我认同混乱
- **重度抑郁/双相情感障碍患者**：长期与死亡念头共存的人
- **自伤行为人群**：通过自伤缓解情绪痛苦的人
- **社会孤立者**：长期缺乏社交连接，感到被世界抛弃的人

#### 用户特征
- 对传统"正能量"文案反感甚至厌恶（"加油""振作"会让他们更难受）
- 对"被理解"有强烈渴望，但讨厌被"同情"
- 熟悉Z世代网络文化，懂梗、会自嘲
- 对"抽象""发疯文学""摆烂"有共鸣
- 在"想死"和"还想活"之间反复横跳
- 深夜是情绪最脆弱、自杀意念最强的时候

#### 为什么聚焦这个群体？
1. **需求最迫切**：他们是心理健康服务覆盖最差、求助意愿最低的群体
2. **现有方案无效**：传统心理咨询太贵、太正式；正能量内容让他们更反感
3. **幽默是最后的防线**：很多自杀幸存者回忆，最后一刻让他们停下来的往往不是大道理，而是一个荒谬的念头、一个搞笑的瞬间
4. **Z世代懂Z世代**：用他们自己的语言、自己的梗、自己的幽默感，建立真实的连接

### 1.5 参赛赛道
「Hello AI 科技致善」公益赛道 - 其他公益方向（心理健康陪伴/自杀预防）

## 2. 需求分析

### 2.1 用户痛点

| 痛点 | 具体场景 | 现有解决方案不足 |
|------|----------|-----------------|
| "想死"的时候不知道能干嘛 | 凌晨3点，躺在床上，脑子里只有"不想活了" | 热线要说话、医生要预约、正能量内容让人更烦 |
| 讨厌被"同情"和"教育" | 朋友安慰"想开点"，反而更想死 | 传统心理支持太正式、太说教 |
| 需要被"懂"而不是被"救" | 发一条"想死"的朋友圈，收到的关心让人窒息 | 没人理解这种"想死"其实是一种求救 |
| 幽默感是最后的防线 | 在自杀笔记里写段子的人 | 没有平台把"黑色幽默"当作疗愈工具 |
| 深夜是最危险的时候 | 凌晨1-5点，自杀意念最强 | 深夜没有人在线，没有资源可用 |
| 对"标签"极度敏感 | 被诊断为"抑郁症"后更想死 | 医学标签让他们感到被定义、被框死 |

### 2.2 用户场景

**场景一：凌晨3点的"不想活了"**
> 小李凌晨3点醒来，脑子里只有一个念头："活着真没意思"。他打开 Soul Soother，看到一个测试"测测你的精神状态有多美丽"，题目包括"你最近一次崩溃是因为？A. 外卖迟到了 B. 发现自己还活着 C. 以上都是"。他笑了，然后哭了，然后觉得"好像还能再活一天"。

**场景二：自伤前的"缓冲"**
> 小张拿起刀片前，习惯性地打开 Soul Soother。今天推荐的是一个"发疯文学生成器"，输入"我想死"，生成了一段："我想死，但我的快递还没到，我的剧还没追完，我的猫还没喂，所以我决定先活着，等这些都做完了再死——结果发现，永远做不完，所以我永远死不了，真抽象。"他放下刀片，笑了。

**场景三：情绪崩溃后的"自我嘲讽"**
> 小王刚大哭完一场，眼睛肿得像核桃。他打开 Soul Soother 的"今日抽象"，看到一句话："我的精神状态：一边想死，一边怕死，一边觉得自己不配死，一边觉得死了也挺好，最后决定先活着，因为死了就看不到这么抽象的自己了。"他截图发朋友圈，收获了一堆"哈哈哈"，突然觉得没那么孤单了。

**场景四：医院排队时的"解压"**
> 小赵在精神科候诊，周围都是沉默的人。他打开 Soul Soother 的"精神病院模拟器"，做了一个测试"如果你被送进精神病院，你会？A. 证明自己没病 B. 发现大家都没病 C. 决定不走了，这里的人比外面正常"。他在候诊室笑出了声，旁边的人也凑过来看，两个人一起笑。

## 3. 核心设计理念："真抽象"的疗愈

### 3.1 什么是"真抽象"？

"真抽象"不是无厘头，而是一种**深刻的自嘲**——
- 承认生活的荒谬性
- 用幽默消解痛苦
- 在"发疯"中找到共鸣
- 不否认痛苦，但不让痛苦定义自己

就像憨豆先生：他不是不知道世界很糟糕，他只是选择用一种荒谬的方式去面对。

### 3.2 内容风格指南

#### ✅ 要做：
- **自嘲式幽默**："我的精神状态：稳定地不稳定"
- **荒诞对比**："别人：事业有成；我：成业有事（成天有事）"
- **黑色幽默**："我想死，但我的外卖还没到，所以我决定先活着"
- **发疯文学**："啊啊啊啊啊我精神状态很好啊哈哈哈哈哈哈"
- **摆烂美学**："累了，毁灭吧，但先让我躺会儿"
- **梗文化**："确诊了，是'想死但不敢'综合征"
- **温柔反击**："世界以痛吻我，我痛死"

#### ❌ 不要做：
- "加油，你可以的！"（让人更想死）
- "想想你的家人"（增加 guilt）
- "明天会更好"（虚假希望）
- "你要坚强"（否定脆弱）
- 任何说教、建议、评判

### 3.3 语言风格示例

| 场景 | 传统方式（❌） | Soul Soother 方式（✅） |
|------|--------------|------------------------|
| 用户说"我想死" | "不要这样想，生命很宝贵" | "想死很正常，我每天都想死，但我的快递还没到，所以先活着" |
| 用户情绪崩溃 | "深呼吸，放松" | "崩溃就崩溃吧，谁还没个发疯的时候，来，一起啊啊啊啊啊" |
| 用户感到孤独 | "你并不孤单" | "确实挺孤单的，但你看，还有我这么抽象的陪你" |
| 用户自伤后 | "不要再伤害自己了" | "又自伤了？没事，我也干过，来，看看这个'自伤替代方案生成器'" |
| 深夜陪伴 | "晚安，好梦" | "晚安，希望你明天还活着——不是诅咒，是真心希望，因为死了就看不到这么抽象的世界了" |

## 4. 功能规划

### 4.1 功能模块

```
Soul Soother
├── 首页 (Home)
│   ├── 今日抽象（每日一句发疯文学）
│   ├── 精神状态检测器（趣味测试）
│   ├── 发疯文学生成器
│   └── 深夜急救包（凌晨1-5点特殊内容）
├── 趣味测评 (Quiz)
│   ├── 测测你的精神状态有多美丽
│   ├── 你的崩溃类型是什么？
│   ├── 如果你被送进精神病院...
│   └── 你的"想死"指数（黑色幽默版）
├── 治愈合集 (Collection)
│   ├── 抽象文案馆（发疯文学、摆烂语录）
│   ├── 自嘲疗愈室（黑色幽默段子）
│   ├── 深夜陪伴站（凌晨专属内容）
│   └── 发疯工具箱（互动生成器）
├── 互动工具 (Tools)
│   ├── 发疯文学生成器
│   ├── 精神状态检测器
│   ├── "想死"替代方案生成器
│   └── 崩溃倒计时（反向鼓励）
└── 关于与安全 (About)
    ├── 项目介绍
    ├── 安全声明
    └── 求助资源（不显眼但可找到）
```

### 4.2 核心功能详解

#### 4.2.1 首页 - "今日抽象"
- **每日一句**：每天更新一句发疯文学/抽象文案
- **示例**：
  - "我的精神状态：一边想死，一边怕死，一边觉得自己不配死，最后决定先活着，因为死了就看不到这么抽象的自己了"
  - "确诊了：我是'想死但不敢'综合征晚期患者，病程：从出生到现在"
  - "别人：内卷；我：内耗；别人：躺平；我：躺尸；别人：发疯；我：稳定地发疯"

#### 4.2.2 精神状态检测器
- **趣味测试**：用荒诞的题目"检测"精神状态
- **题目示例**：
  - "你最近一次崩溃是因为？A. 外卖迟到了 B. 发现自己还活着 C. 以上都是 D. 我不崩溃，我直接发疯"
  - "你的睡眠状况？A. 睡得很好 B. 睡不着 C. 不想睡 D. 睡了但感觉像没睡，因为梦里也在想死"
  - "你的社交状态？A. 很多朋友 B. 有几个朋友 C. 没有朋友 D. 有，但都是网友，而且我们互相不知道真名"
- **结果示例**：
  - "你的精神状态：美丽且抽象。建议：继续保持，毕竟正常人才是真的疯了"
  - "你的崩溃类型：'表面平静内心海啸'型。建议：找个没人的地方大喊一声，或者来我们这里发疯"

#### 4.2.3 发疯文学生成器
- **输入情绪关键词**：如"想死""崩溃""孤独"
- **生成发疯文学**：
  - 输入"想死" → "我想死我想死我想死但我的快递还没到我的剧还没追完我的猫还没喂所以我决定先活着等这些都做完了再死结果发现永远做不完所以我永远死不了真抽象哈哈哈哈"
  - 输入"崩溃" → "崩溃就崩溃吧谁还没个发疯的时候啊啊啊啊啊我精神状态很好啊哈哈哈哈哈哈世界毁灭吧但先让我躺会儿"

#### 4.2.4 "想死"替代方案生成器
- **输入**："我想死"
- **生成替代方案**：
  - "先吃完这顿外卖再死"
  - "等这部剧更新完再死"
  - "先把花呗还完再死"
  - "等猫找到新主人再死"
  - "先看完明天的日出再死——然后发现每天都能看到日出，所以每天都不能死"

#### 4.2.5 深夜急救包（1:00-5:00特殊内容）
- **凌晨专属内容**：
  - "凌晨3点还没睡？恭喜你，你正在经历'深夜哲学家'模式"
  - "凌晨4点的想法不是真实的，是你的大脑在骗你"
  - "再坚持3小时，太阳就出来了——虽然太阳出来也不一定能解决问题，但至少天亮了"
  - "给凌晨的你：你现在想的'不想活了'，天亮后可能变成'不想上班'，虽然也很痛苦，但至少不致命"

### 4.3 内容分类

#### 抽象文案馆
- **发疯文学**：啊啊啊啊啊我精神状态很好啊哈哈哈哈
- **摆烂语录**：累了，毁灭吧，但先让我躺会儿
- **自嘲段子**：我的优点：知错就改；我的缺点：改了再犯
- **黑色幽默**：我想死，但我的外卖还没到

#### 自嘲疗愈室
- **废物文学**：这世界总有人要当废物，那为什么不能是我？
- **社恐语录**：别人：社交牛逼症；我：社交恐惧症；实际：社交不存在症
- **穷鬼段子**：我的钱包比我的脸还干净
- **单身自嘲**：别人：脱单；我：脱发

#### 深夜陪伴站
- **凌晨专属**：凌晨3点的你不是一个人，还有我们这群夜猫子
- **失眠陪伴**：睡不着就睡不着吧，反正明天也不用早起——因为明天也不想活
- **噩梦安抚**：噩梦醒了？没事，现实可能比噩梦还抽象

#### 发疯工具箱
- **发疯文学生成器**
- **精神状态检测器**
- **"想死"替代方案生成器**
- **崩溃倒计时**："距离你上次崩溃已经过了XX小时，恭喜你，创造了新纪录"

#### 发疯文学模板

```
[别人在做的事]：[积极结果]
[我在做的事]：[消极但搞笑的结果]
结论：[摆烂宣言]
```

#### 黑色幽默模板
```
[极端想法]，但[荒谬的阻碍]，所以[荒谬的结论]
```

### 5.3 内容安全红线

#### 绝对禁止
- ❌ 详细描述自杀方法
- ❌ 美化或鼓励自杀
- ❌ 展示自伤图片
- ❌ 歧视任何群体
- ❌ 涉及具体人物的恶意攻击

#### 需要谨慎处理
- ⚠️ 自杀相关话题：可以用幽默消解，但不能轻视
- ⚠️ 自伤相关话题：可以提供替代方案，但不能鼓励
- ⚠️ 精神疾病标签：可以自嘲，但不能污名化

#### 推荐做法
- ✅ 用"想死"作为幽默素材，但始终暗示"活着也挺好"
- ✅ 承认痛苦的合理性，但用荒谬的方式解构
- ✅ 在搞笑中隐藏温暖，让用户在笑完后感到被理解

## 7. 设计规范

### 7.1 色彩系统 - Cocoa Ink Palette

**设计哲学**：以可可棕与羊皮纸为基调，营造温暖、有质感的阅读氛围，避免冷冰冰的"医疗感"。

```css
:root {
  /* 主色调 - Cocoa Ink */
  --color-ink: #472425;          /* 深可可色：主文字色，温暖而沉稳 */
  --color-charcoal: #472425;     /* 与 Ink 一致，正文强调色 */
  --color-cocoa: #80705f;        /* 可可棕：次要文字、引力场粒子 */
  --color-slate: #878887;        /* 石板灰：辅助说明、标签文字 */

  /* 画布背景 - Parchment */
  --color-canvas: #fdfaf3;       /* 羊皮纸白：主背景，温暖不刺眼 */
  --color-canvas-warm: #f5f0e8;  /* 暖羊皮纸：卡片悬停、输入框背景 */

  /* 边框与分隔 */
  --color-border: #e8e0d4;       /* 浅可可边框：卡片、按钮边框 */
  --color-border-subtle: #f0ebe3;/* 极淡边框：分隔线 */

  /* 功能色 */
```
