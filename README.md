# Soul Soother - 心灵抚慰者

> 用抽象接住崩溃，用玩笑降低痛感

## 项目简介

Soul Soother 是一个面向极端情绪状态人群（自杀意念高发群体）的轻量级趣味治愈网站。我们不提供医学诊断，也不替代心理咨询，而是用 Z 世代最懂的"抽象自嘲"语言，让濒临崩溃的人在会心一笑中获得短暂的喘息和陪伴。

## 核心功能

| 页面           | 功能                                                         |
| -------------- | ------------------------------------------------------------ |
| 🏠 **首页**     | p5.js 语义引力场 + 紧急入口 + 免责声明                       |
| 💓 **快速检测** | "小缓"对话式状态检测                                         |
| 🤝 **陪伴**     | 陪伴卡抽取 + 深夜弹幕墙 + 留一句给陌生人                     |
| 🛟 **缓冲站**   | 30秒呼吸空间 + 缓冲卡（轻度/中度/高危）+ 温暖资源            |
| 🎈 **缓一缓**   | 7种自助工具——喘口气/抓回现实/等一下/找理由/倒垃圾/翻译情绪/小游戏 |

### 缓一缓页面详解

| 工具       | 说明                                                |
| ---------- | --------------------------------------------------- |
| 喘口气     | 跟着圆圈呼吸的呼吸练习                              |
| 抓回现实   | 3-2-1 感官着陆法（看3样/听2种/摸1个）               |
| 等一下     | 生成"先做完XX再死"的荒谬列表                        |
| 找理由     | 抽取"还有什么舍不得"的依恋卡                        |
| 倒垃圾     | 输入情绪关键词生成发疯文学                          |
| 翻译情绪   | 坏念头翻译机（"想死"→"先离开当前场景，去倒一杯水"） |
| 玩个小游戏 | 推荐治愈网页和小游戏                                |

## 彩蛋系统

✨ **首页连击系统**：连续点击彩色词汇触发递进反馈（"抓住了一点光"→"光之使者"）

🐱 **"想死"替代方案彩蛋**：15种主题（猫咪安慰/狗狗鼓励/美食诱惑等），全局5%触发概率，深夜15%概率

🕐 **时间敏感彩蛋**：深夜猫头鹰🦉、凌晨幽灵👻、早晨朝阳🌅、工作时间咖啡☕、傍晚夕阳🌇

🎂 **特殊日期彩蛋**：生日🎂、新年🎆

🚪 **隐藏入口**：连点10次"Built with TRAE SOLO"→AI设置，长按800ms→记忆管理

## 技术栈

- 纯 HTML5 + CSS3 + JavaScript（零框架，极致轻量）
- p5.js 文字引力场
- Cocoa Ink 设计系统（可可棕+羊皮纸白+薰衣草紫）
- Google Fonts（ZCOOL XiaoWei + Noto Serif SC）
- localStorage（仅本地存储对话记录）
- GitHub Pages 部署

## 设计特点

- **零门槛**：无需注册、无需登录、打开即用
- **低认知负荷**：大按钮、少选项、"我现在想..."快速选择
- **无压迫感**：没有待办、没有进度、没有通知、没有社交比较
- **状态导向**：不是功能列表，是情绪状态的引导路径
- **抽象语言**：用 Z 世代最懂的梗，不是"加油"是"阴暗爬行也算移动"
- **安全边界**：免责声明、缓冲卡分级、危机内容引导到求助资源
